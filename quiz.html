Linux Quiz

**Q1.** Who created Linux?

A) Bill Gates
B) Steve Jobs
C) Linus Torvalds
D) Mark Zuckerberg

Answer: C) Linus Torvalds

**Q2.** What is the core part of the Linux operating system called?

A) Shell
B) Kernel
C) GUI
D) Command-line

Answer: B) Kernel

**Q3.** Which of the following is NOT a feature of Linux?

A) Open-source
B) Free to use
C) Highly customizable
D) Requires a paid license

Answer: D) Requires a paid license

**Q4.** What is one of the main disadvantages of using Windows compared to Linux?

A) It is open-source.
B) It is highly customizable.
C) It requires frequent license payments.
D) It is virus-free.

Answer: C) It requires frequent license payments.

**Q5.** Why is Windows more prone to security vulnerabilities?

A) It is open-source.
B) It uses a robust permissions model.
C) It has a large user base and relies on third-party antivirus.
D) It has fewer users.

Answer: C) It has a large user base and relies on third-party antivirus.

**Q6.** Which of the following is a benefit of using Linux?

A) High cost
B) Frequent security vulnerabilities
C) Customizable system
D) Limited hardware compatibility

Answer: C) Customizable system

**Q7.** Why is Linux considered more secure than Windows?

A) It’s rarely updated.
B) It allows users to modify and view source code.
C) It has weak permission models.
D) It is a closed-source system.

Answer: B) It allows users to modify and view source code.

**Q8.** What makes Linux more stable compared to some other operating systems?

A) It has no users.
B) It is open to public modification and audit.
C) It is available for free.
D) It is closed-source.

Answer: B) It is open to public modification and audit.

**Q9.** What is proprietary software?

A) Software that is free to use and modify.
B) Software whose source code is owned and controlled by an individual or company.
C) Software licensed under the GPL.
D) Open-source software.

Answer: B) Software whose source code is owned and controlled by an individual or company.

**Q10.** What does the GPL license require from developers who modify and distribute GPL-licensed software?

A) They must keep the changes private.
B) They must re-license the software as proprietary.
C) They must make the modified code available under the GPL.
D) They must pay a licensing fee.

Answer: C) They must make the modified code available under the GPL.

**Q11.** What is GNU in the context of GNU/Linux?

A) A proprietary software organization
B) A software company owned by Microsoft
C) A set of free software tools that work with the Linux kernel
D) The kernel of the Linux operating system

Answer: C) A set of free software tools that work with the Linux kernel.

**Q12.** What does GNU stand for?

A) General Network Unit
B) GNU’s Not Unix
C) Global Network Utility
D) Great Network Utility

Answer: B) GNU’s Not Unix.

**Q13.** Which of the following is a Linux distribution known for being beginner-friendly?

A) Arch Linux
B) Ubuntu
C) Debian
D) Gentoo

Answer: B) Ubuntu.

**Q14.** What is the primary focus of the Fedora distribution?

A) Cutting-edge features and innovation
B) Enterprise server stability
C) Minimalist design
D) Education and research

Answer: A) Cutting-edge features and innovation.

**Q15.** Which of the following is a common use for Linux?

A) Desktop computing
B) Web servers
C) Embedded systems
D) All of the above

Answer: D) All of the above.

**Q16.** Which type of device commonly runs Linux?

A) Windows desktop
B) Apple iPhones
C) Android smartphones
D) Sony PlayStation

Answer: C) Android smartphones.

**Q17.** Which of the following file systems is commonly used by Linux?

A) NTFS
B) FAT32
C) ext4
D) APFS

Answer: C) ext4.

**Q18.** What is the primary way Linux users manage their systems?

A) Through a graphical interface
B) Through the terminal and command-line interface (CLI)
C) Through Windows File Explorer
D) Through BIOS settings

Answer: B) Through the terminal and command-line interface (CLI).

**Q19.** Which package manager does Ubuntu use for installing and managing software?

A) Yum
B) Pacman
C) Apt
D) RPM

Answer: C) Apt.

**Q20.** Ubuntu is a good choice for new users because:

A) It is a proprietary system
B) It is very difficult to use
C) It has a user-friendly interface and good community support
D) It is expensive

Answer: C) It has a user-friendly interface and good community support.
