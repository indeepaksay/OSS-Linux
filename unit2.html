Introduction to BASH Shell

BASH (Bourne Again SHell) is a command-line interpreter that provides a command line interface for interacting with the operating system. It is the default shell for many Linux distributions and macOS, allowing users to execute commands, manage files, and automate tasks through scripting.

Key Features of BASH

1. Command History

BASH keeps a history of commands you've executed, which you can easily access.

Example: Pressing the up arrow key retrieves the last command you entered.

2. Job Control

BASH allows users to run multiple processes in the background or foreground.

Example:

  • To run a command in the background, append &:
    sleep 10 &
  • To bring a background job to the foreground, use the fg command.

3. Scripting

You can write scripts to automate tasks. A BASH script is a file containing a series of commands.

Example: Create a script hello.sh:

#!/bin/bash\necho "Hello, World!"

Make it executable and run:

chmod +x hello.sh
./hello.sh

Common BASH Commands

1. Navigating the File System

Listing Files:

ls

Change Directory:

cd Documents

2. File Manipulation

Copy Files:

cp file.txt backup_file.txt

Move/Rename Files:

mv oldname.txt newname.txt

Delete Files:

rm file.txt

3. Directory Management

Create a Directory:

mkdir new_directory

Remove a Directory:

rmdir empty_directory

4. Viewing File Contents

Display Contents:

cat file.txt

View File with Pagination:

less file.txt

5. Getting Help

Most commands come with built-in help. Use man (manual) for detailed documentation.

Example:

man ls

6. Using Variables

You can store values in variables.

Example:

name="Alice"
echo "Hello, $name!"

BASH is a powerful tool for interacting with the operating system, offering extensive features for file management, scripting, and system control. Mastering BASH commands and scripting can significantly enhance productivity and streamline workflows in a Unix-like environment.

The BASH (Bourne Again SHell) is a command-line interpreter used in many Linux distributions and macOS. It allows users to interact with the operating system by executing commands and scripts. It supports features like command history, job control, and scripting capabilities.

Working with Files

BASH provides numerous commands for file manipulation, allowing you to manage files and directories effectively.

Listing Files

Command: ls

Example:

ls

This command lists files in the current directory.

With options:

ls -l

This command lists files in long format, showing permissions, ownership, file size, and modification date.

Copying Files and Directories

Command: cp

Example:

cp source.txt destination.txt

This command copies source.txt to destination.txt.

Copying a directory:

cp -r source_dir/ destination_dir/

The -r option is used to copy directories recursively.

Moving Files and Directories

Command: mv

Example:

mv oldname.txt newname.txt

This command renames oldname.txt to newname.txt.

Moving a file to a different directory:

mv file.txt /path/to/directory/

Deleting Files and Directories

Command: rm

Example:

rm file.txt

This command deletes file.txt.

Deleting a directory:

rm -r directory_name/

Use -r to delete directories and their contents.

Changing and Creating Directories

Creating a directory:

Command: mkdir

Example:

mkdir new_directory

Changing the current directory:

Command: cd

Example:

cd new_directory

Real Files and Virtual Files

Real files: These are standard files stored on disk.

Virtual files: These are not real files but represent system information (e.g., /proc, /sys).

Users and File Permissions

Viewing file permissions:

Command: ls -l

Example:

ls -l

This will show the permissions of files, formatted as rwxr-xr-x.

Changing permissions:

Command: chmod

Example:

chmod 755 script.sh

The File System Explained

The Linux file system is hierarchical. It starts at the root (/) and branches out to directories and files. Common directories include:

  • /home: User home directories
  • /etc: Configuration files
  • /var: Variable files, like logs

File Searches

BASH provides several commands to search for files.

Using the find Command

Command: find

Example:

find /path/to/search -name "*.txt"

This searches for all .txt files in the specified directory.

Using the locate Command

Command: locate

Example:

locate filename

This finds files by name using a database that is updated periodically.

Using the whereis Command

Command: whereis

Example:

whereis bash

This finds the binary, source, and manual page for bash.

File Size and Free Space

You can check file sizes and free disk space with several commands.

Viewing File Sizes

Command: du

Example:

du -h filename

This displays the size of the specified file or directory in a human-readable format.

Finding Out the Amount of Free Space

Command: df

Example:

df -h

This shows disk space usage for all mounted filesystems in a human-readable format.
